# 觅阅 MiuRead：KOReader 微信读书插件
觅阅（MiuRead）是一款适用于 Kindle 和 KOReader 的非官方微信读书（WeRead）插件，支持微信读书书架浏览、书籍下载、划线与想法、阅读时长同步和阅读进度同步。

## 当前状态

项目仍处于开发测试阶段，部分功能可能继续调整。

当前主要功能包括：

- 微信读书账号登录
- 微信读书书架浏览
- 书籍搜索
- 纯净版书籍下载
- 划线与想法版书籍下载
- 章节单独下载
- 热门划线与想法查看
- 阅读时间同步
- 阅读进度同步
- Kindle Voyage 实体翻页键支持
- 插件在线更新

## 支持环境

主要测试环境：

- Kindle Voyage
- KOReader

其他 Kindle 或 KOReader 设备可能可以使用，但尚未完整测试。
