# KOReader Patches

A collection of user patches for [KOReader](https://github.com/koreader/koreader).

## Patches

| Patch | Description |
|-------|-------------|
| [Browser Folder Cover](2-browser-folder-cover/) | Display folder covers in Mosaic view — from custom `.cover` images or book covers found inside the folder |
| [Clean Orphaned SDR](2-clean-orphaned-sdr/) | Automatically remove orphaned `.sdr` sidecar folders on startup |
| [Sort by Author-Series](2-sort-by-author-series/) | Add four sorting modes: author → series → title/date |

## Installation

1. Download the `.lua` patch file from the patch folder
2. Copy it to your KOReader patches directory:
   - Kobo: `/mnt/onboard/.adds/koreader/patches/`
   - Kindle: `/mnt/us/documents/koreader/patches/`
   - Android: `/sdcard/koreader/patches/`
   - Desktop: `~/.koreader/patches/`
3. Restart KOReader

Create the `patches` folder if it doesn't exist. For more details, see the [KOReader User Patches wiki](https://github.com/koreader/koreader/wiki/User-patches).
