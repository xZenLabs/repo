# fanqie.koplugin

KOReader 插件，用于在电纸书设备上阅读番茄小说。

## 功能特性

- 📚 **书架管理**：浏览和搜索番茄小说，支持封面显示
- 📖 **章节阅读**：支持章节导航、自动翻页、目录自动跳转
- 🔄 **进度同步**：自动同步阅读进度到服务器
- ⚡ **预下载**：智能预下载后续章节，减少阅读等待
- 🎨 **界面优化**：适配墨水屏显示，支持黑白模式
- 📊 **阅读统计**：显示阅读进度（百分比/章节数）
- 📥 **批量下载**：支持下载全书，自动跳过已缓存章节
- 🔀 **书架排序**：支持按阅读进度、最后阅读时间、加入时间、书名排序

## 安装方法

1. 下载插件压缩包（从 GitHub Releases 页面）
2. 将 `fanqie.koplugin` 文件夹复制到 KOReader 的 `plugins/` 目录
3. 将 `config.example.lua` 复制一份，改名为 `config.lua`
4. 在 `config.lua` 中配置 `cookie_string`（获取方法见下方）
5. 重启 KOReader
6. 在菜单中找到「番茄书架」入口

## Cookie 获取方式

### 方法一：浏览器开发者工具（推荐）

1. 打开浏览器（推荐 Chrome 或 Edge）
2. 访问 [番茄小说网页版](https://fanqie.com)
3. 登录你的账号
4. 按下 `F12` 打开开发者工具
5. 切换到「Network」（网络）标签页
6. 在页面上点击「书架」或刷新页面，触发网络请求
7. 在网络请求列表中找到 `multidetail` 请求（URL 类似 `https://fanqienovel.com/api/bookshelf/multidetail?...`），点击查看详情
8. 在「Headers」（请求头）中找到 `Cookie` 字段
9. 右键点击 `Cookie` 值，选择「Copy value」（复制值）
10. 将复制的完整 Cookie 字符串粘贴到插件配置中

### 方法二：浏览器扩展

安装 Cookie 导出插件（如「EditThisCookie」），一键导出 Cookie 字符串。

## 配置说明

将 `config.example.lua` 复制一份，改名为 `config.lua`，然后根据需要修改配置项：

```lua
return {
    -- Cookie 字符串（必填）
    cookie_string = "serial_uuid=xxx; sessionid=xxx; ttwid=xxx; ...",
    
    -- 下载目录（默认使用插件目录下的 downloads 文件夹）
    download_dir = nil,
    
    -- 预下载章节数（默认 3 章）
    pre_download_chapters = 3,
    
    -- 是否在打开书籍时同步进度（默认 true）
    sync_pull_on_open = true,
    
    -- 是否在翻页时同步进度（默认 true）
    sync_on_page_update = true,
}
```

## 使用指南

### 书架操作

- **点击书籍**：打开书籍详情页面
- **点击「刷新书架」**：重新获取书架数据
- **长按书籍**：显示操作菜单（开始阅读、章节目录、下载全书、清空缓存）

### 阅读操作

- **点击左侧**：上一页
- **点击右侧**：下一页
- **点击中间**：打开菜单
- **长按屏幕**：显示菜单选项
- **章节开头向前翻页**：自动跳转到上一章
- **章节结尾向后翻页**：自动跳转到下一章

### 目录操作

- **打开目录**：自动跳转到当前阅读章节所在页面
- **点击章节**：跳转到对应章节
- **缓存标记**：已下载的章节显示 ✓ 标记

### 下载全书

- 支持选择下载范围（全部、前10章、自定义范围）
- 已缓存的章节会自动跳过，避免重复下载
- 下载过程中可以取消

本插件使用的正文获取接口为第三方公开接口，**非官方 API**。

### 使用规范

1. **请勿一次性下载全本书**：插件默认仅预下载 3 章，请保持此设置

## 常见问题

### Q: 书架显示为空？

A: 请检查 `config.lua` 中的 Cookie 是否正确，Cookie 过期会导致无法获取书架数据。

### Q: 章节下载失败？

A: 可能是网络问题或接口限制，请稍后重试。如果问题持续，请检查 Cookie 是否有效。

### Q: 阅读进度不同步？

A: 请确保网络连接正常，进度同步需要网络支持。

### Q: 封面不显示？

A: 封面是异步下载的，首次打开书架时需要等待一段时间。如果长时间不显示，请检查网络连接。
