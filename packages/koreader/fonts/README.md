# KOReader fonts

This directory caches the e-reader-optimized font families published by
[nicoverbruggen/ebook-fonts](https://github.com/nicoverbruggen/ebook-fonts).
Each family has its own folder containing metadata, a locally cached preview
image, and an installable ZIP with only that family's styles.

`ebook-fonts-all.zip` contains every cached family under a single `fonts/`
directory, ready to unpack into a KOReader installation. It is regenerated by
the scheduled font scraper from the upstream release manifest and archives.

Each font package credits `nicoverbruggen`; the font files retain their
upstream embedded copyright and license information.
